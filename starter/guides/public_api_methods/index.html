Публичное API

POST /services/{serviceId}/tasks

Добавляет задачу в очередь

Пример вызова

curl -X POST 'http://build.myapp.lan:28346/services/Core.ImportAccountAndCampaignsYandexDirect/tasks' -H 'Accept: application/json' -H 'Content-Type: application/json' --data '{"serviceId":"Core.ImportAccountAndCampaignsYandexDirect", "data":{"agencyLocalId":381}}'

serviceId - имя зарегистрированного сервиса data - пользователький набор данных для задачи

GET /configurations

Отдает все настройки сервисов

POST /configurations

Заставляет перезагрузить конфигурацию

GET /services/{serviceId}/statistics

Возвращает информацию по статистике использования сервиса с последнего запуска
